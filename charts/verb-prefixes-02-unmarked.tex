\section{Unmarked aspect: imperfective or \textit{∅}-conjugation prefix}\label{sec:zero}

\begin{multicols}{2}
\noindent
Unmarked aspect is any form without an overt aspect prefix, i.e.\ a form without a perfective prefix (\fm{\Af{wu}-} or \fm{\Af{u}-}) and without an overt conjugation prefix (\fm{\Af{n}-}, \fm{\Af{g̱}-}, or \fm{\Af{g}-}).
The simplest cases of unmarked aspect are imperfective aspect forms and imperative mood forms.
The imperfective aspect form of most activity and state verbs has no aspect marking as shown by the examples of an imperfective activity and an imperfective state in (\ref{ex:zero-imperfectives}).

\pex\label{ex:zero-imperfectives}%
\a\label{ex:zero-imperfectives-activity}%
\exrtcmt{∅-imperfective activity}%
\begingl
	\gla	\rlap{\Sf{Yi}x̱á.} @ {} @ {} @ {} //
	\glb	{} \Sf{yi}- \rt[²]{x̱a} -μH //
	\glc	{} \Sf{\xx{2pl·s}}- \rt[²]{eat} -\xx{var} //
	\gld	\rlap{3.\xx{impfv}.you·\xx{pl}.eat} {} {} {} //
	\glft	‘You guys eat him/her/it.’, ‘You guys are eating him/her/it.’
		//
\endgl
\a\label{ex:zero-imperfectives-state}%
\exrtcmt{∅-imperfective state}%
\begingl
	\gla	\rlap{\Qf{I}\Sf{tu}\If{w}\Ef{a}téen.} @ {} @ {} @ {} @ {} //
	\glb	\Qf{i}- \Sf{tu}- \If{i}- \rt[²]{tin} -μH //
	\glc	\Qf{\xx{2sg·o}}- \Sf{\xx{1pl·s}}- \If{\xx{stv}}- \rt[²]{see} -\xx{var} //
	\gld	\rlap{you·\xx{sg}.\xx{impfv}.we.can·see} {} {} {} {} {} //
	\glft	‘We can see you.’
		//
\endgl
\xe

Imperfective aspect forms like those in (\ref{ex:zero-imperfectives}) have no overt prefix for aspect since they have neither a perfective prefix \fm{\Af{wu}-} or \fm{\Af{u}-} nor an overt conjugation prefix \fm{\Af{n}-}, \fm{\Af{g̱}-}, or \fm{\Af{g}-}.
It is possible to analyze imperfectives as having a \fm{\Af{∅}} conjugation prefix, but it is equally possible to analyze them as having no prefix whatsoever.
Both analyses give rise to the same verb forms so for verb prefix phonology and verb paradigm patterns the difference between these two analyses is irrelevant.

The other simple case of unmarked aspect is the imperative mood form of verbs that belong to the \fm{∅}-conjugation class.
Imperative mood selects a prefix corresponding to the conjugation class membership of the verb.
Thus a verb belonging to the \fm{n}-conjugation class will contain a \fm{\Af{n}-} conjugation prefix, and similarly a verb of the \fm{g}-conjugation class will contain a \fm{\Af{g}-} conjugation prefix.
For a \fm{∅}-conjugation class verb the imperative simply lacks a conjugation prefix.
This lack of a prefix is analyzed as the \fm{\Af{∅}} conjugation prefix.
Compare the \fm{∅}-conjugation class imperative in (\ref{ex:zero-imperatives-z}) with the other three conjugation class imperatives in (\ref{ex:zero-imperatives-n})–(\ref{ex:zero-imperatives-g}).

\pex\label{ex:zero-imperatives}%
\a\label{ex:zero-imperatives-z}%
\exrtcmt{∅-conjugation imperative}%
\begingl
	\gla	\rlap{\Sf{Yi}x̱á!} @ {} @ {} @ {} //
	\glb	\Af{∅} \Sf{yi}- \rt[²]{x̱a} -μH //
	\glc	\Af{\xx{zcnj}}\· \Sf{\xx{2pl·s}}- \rt[²]{eat} -\xx{var} //
	\gld	\rlap{3.\xx{imp}.you·\xx{pl}.eat} {} {} {} //
	\glft	‘You guys eat him/her/it!’
		//
\endgl
\a\label{ex:zero-imperatives-n}%
\exrtcmt{n-conjugation imperative}%
\begingl
	\gla	\rlap{\Af{N}\Ef{a}\Sf{y}hoon!} @ {} @ {} @ {} //
	\glb	\Af{n}- \Sf{yi}- \rt[²]{hun} -μμL //
	\glc	\Af{\xx{ncnj}}- \Sf{\xx{2pl·s}}- \rt[²]{sell} -\xx{var} //
	\gld	\rlap{3.\xx{imp}.you·\xx{pl}.sell} {} {} {} //
	\glft	‘You guys sell him/her/it!’
		//
\endgl
\a\label{ex:zero-imperatives-gh}%
\exrtcmt{g̱-conjugation imperative}%
\begingl
	\gla	\rlap{\Af{G̱}\Ef{a}\Sf{y}x̱oox̱!} @ {} @ {} @ {} //
	\glb	\Af{g̱}- \Sf{yi}- \rt[²]{x̱ux̱} -μμL //
	\glc	\Af{\xx{g̱cnj}}- \Sf{\xx{2pl·s}}- \rt[²]{summon} -\xx{var} //
	\gld	\rlap{3.\xx{imp}.you·\xx{pl}.summon} {} {} {} //

		//
\endgl

\exrtcmt{g-conjugation imperative}%
\begingl
	\gla	\rlap{\Af{G}\Ef{a}\Sf{y}shí!} @ {} @ {} @ {} //

\subsection{Unmarked aspect with irrealis \textit{u-}}\label{sec:zero-irrealis}

\begin{multicols}{2}
\noindent
The irrealis prefix \fm{\Rf{u}-} can occur in unmarked aspect forms.
One situation that requires irrealis \fm{\Rf{u}-} is negation with the negative particle \fm{tléil} ‘not’ or its various allomorphs.
Compare the affirmative imperfective activity in (\ref{ex:zero-irrealis-negation-aff}) below with the negative in (\ref{ex:zero-irrealis-negation-neg}).

\pex\label{ex:zero-irrealis-negation}%
\a\label{ex:zero-irrealis-negation-aff}%
\exrtcmt{affirmative}%
\begingl
	\gla	\rlap{\Qf{A}x̱á.} @ {} @ {} @ {} //
	\glb	\Qf{a}- \rt[²]{x̱a} -μH //
	\glc	\Qf{\xx{arg}}- \rt[²]{eat} -\xx{var} //
	\gld	\rlap{3>3.\xx{impfv}.eat} {} {} {} //
	\glft	‘S/he/it eats it.’, ‘S/he/it is eating it.’
		//
\endgl
\a\label{ex:zero-irrealis-negation-neg}%
\exrtcmt{negative}%
\begingl
	\gla	Tléil \rlap{\Qf{o}\Rf{o}x̱á.} @ {} @ {} @ {} //
	\glb	tléil \Qf{a}- \Rf{u}- \rt[²]{x̱a} -μH //
	\glc	\xx{neg} \Qf{\xx{arg}}- \Rf{\xx{irr}}- \rt[²]{eat} -\xx{var} //
	\gld	not \rlap{3>3.\xx{irr}.\xx{impfv}.eat} {} {} {} //
	\glft	‘S/he/it does not eat it.’, ‘S/he/it is not eating it.’
		//
\endgl
\xe

Another situation that requires irrealis \fm{\Rf{u}-} is when it is specified as part of the lexical entry of a verb.
A typical verb with lexically specified irrealis is \fm{yéi oowajée} (\fm{n}; \fm{-μμH} state impfv.)\ ‘s/he/it thinks so about him/her/it’.
Compare the imperfective aspect form of this verb in (\ref{ex:zero-irrealis-lexical-with}) with its ungrammatical counterpart in (\ref{ex:zero-irrealis-lexical-without}) that lacks irrealis \fm{\Rf{u}-}.

\pex\label{ex:zero-irrealis-lexical}%
\a\label{ex:zero-irrealis-lexical-with}%
\exrtcmt{with irrealis}%
\begingl
	\gla	Yéi @ \rlap{\Qf{o}\Rf{o}\If{w}\Ef{a}jée.} @ {} @ {} @ {} @ {} //
	\glb	yéi= \Qf{a}- \Rf{u}- \If{i}- \rt[²]{jiʰ} -μμH //
	\glc	thus= \Qf{\xx{arg}}- \Rf{\xx{irr}}- \If{\xx{stv}}- \rt[²]{think} -\xx{var} //
	\gld	so \rlap{3>3.\xx{irr}.\xx{impfv}.think} {} {} {} {} //
	\glft	‘S/he/it thinks so about him/her/it’
		//
\endgl
\a\label{ex:zero-irrealis-lexical-without}%
\ljudge{*}%
\exrtcmt{*without irrealis}%
\begingl
	\gla	Yéi @ \rlap{\Qf{a}\If{y}\Ef{a}jée.} @ {} @ {} @ {} @ {} //
	\glb	yéi= \Qf{a}- \phantom{\Rf{u}-} \If{i}- \rt[²]{jiʰ} -μμH //
	\glc	thus= \Qf{\xx{arg}}- \phantom{\Rf{\xx{irr}}-} \If{\xx{stv}}- \rt[²]{think} -\xx{var} //
	\gld	so \rlap{3>3.\xx{impfv}.think} {} {} {} {} //
	\glft	‘S/he/it thinks so about him/her/it’
		//
\endgl
\xe

Irrealis marking with the \fm{\Rf{u}-} prefix is visible only in some unmarked aspect forms and can be undetectable in others.
Irrealis can be reliably detected in forms with an unmarked subject (third person or no subject) and forms with a first person singular subject \fm{\Sf{x̱}-} ‘I’.
Irrealis can sometimes be detected in forms with a fourth person human (indefinite or nonspecific) subject \fm{\Sf{du}-} ‘someone, people, they’ (Leer 1991: 176 \#5a), but this is not always reliable and for some speakers is limited to frozen forms in song lyrics and proverbs.

Irrealis cannot be detected in any unmarked aspect forms with
\begin{inlineenum}
	\item	first person plural subject \fm{\Sf{tu}-} ‘we’
	\item	second person singular subject \fm{\Sf{i}-} ‘you sg.’
	\item	second person plural subject \fm{\Sf{yi}-} ‘you guys’.
\end{inlineenum}
Tables of these forms are given in this subsection for completeness but they are identical to the tables in the previous section.
In such forms where irrealis \fm{\Rf{u}-} is predicted but cannot be detected, we analyze the irrealis prefix as still being logically present but having somehow been hidden.
The reasons for this disappearance of the irrealis prefix are still unclear, but they probably have to do with the phonology of the vowel /\ipa{u}/ and its interactions with nearby /\ipa{i}/ and /\ipa{u}/ vowels in the same form.

Irrealis unmarked aspect forms with the stative classifier prefix \fm{\If{i}-} are underdocumented.
This is because they are only possible with imperfective state verbs where the irrealis \fm{\Rf{u}-} is lexically specified.
In some cases there may simply be no verb with lexically specified irrealis \fm{\Rf{u}-} that takes a subject and includes a qualifier or incorporate prefix.
Such forms are thus grammatically plausible but do not exist, meaning that they are accidental gaps in the language.

There is a remarkable amount of variation in unmarked aspect forms with irrealis \fm{\Rf{u}-} and the first person singular subject \fm{\Sf{x̱}-} ‘I’.
The reasons for this extensive variation are unknown.
Some of this variation is probably specific to particular dialects, but the details of the variation and its geographic distribution have yet to be clearly characterized.
A number of grammatically likely forms are not attested and need to be elicited from speakers of different varieties.
\end{multicols}

\subsection{Unmarked aspect with modal \textit{g̱-}}\label{sec:zero-modal}

\begin{multicols}{2}
\noindent
The modal prefix \fm{\Mf{g̱}-} can occur in unmarked aspect forms.
Imperfective aspect does not occur with the modal prefix \fm{\Mf{g̱}-}.
This means that the only unmarked aspect forms with \fm{\Mf{g̱}-} are \fm{∅}-conjugation class verbs with the combination of \fm{\Af{\xx{cnj}}-} + \fm{\Mf{g̱}-} where \fm{\Af{\xx{cnj}}-} = \fm{\Af{∅}}.
The \fm{\Mf{g̱}-} modal prefix only occurs in
\begin{inlineenum}
	\item	prospective (‘future’) aspect
	\item	potential modality
	\item	hortative mood
	\item	contingent mood.
\end{inlineenum}
The prospective aspect can only be formed with the \fm{\Af{g}-} conjugation prefix and so cannot have \fm{\Af{∅}}.
The potential modality can occur with \fm{\Af{∅}} but it also requires the presence of irrealis \fm{\Rf{u}-} (see section \ref{sec:zero-irrealis+modal}).
These restrictions mean that the only two contexts for forms with just \fm{\Af{∅}} and \fm{\Mf{g̱}-} are the hortative and the contingent of \fm{∅}-conjugation class verbs.

The forms in (\ref{ex:zero-modal-forms}) illustrate this pattern using the \fm{∅}-conjugation verb \fm{ax̱á} (\fm{∅}; \fm{-μH} activity) ‘s/he/it eats him/her/it’.
The imperative in (\ref{ex:zero-modal-forms-imp}) confirms that this is \fm{∅}-conjugation.
The hortative in (\ref{ex:zero-modal-forms-hort}) and the contingent in (\ref{ex:zero-modal-forms-hort}) show the resulting appearance of the argument marking \fm{\Qf{a}-}, the \fm{\Af{∅}} conjugation, and the \fm{\Mf{g̱}-} modal.

The \fm{\Mf{g̱}-} modal prefix is homophonous with the \fm{\Af{g̱}-} conjugation prefix, but they can occur together (sec.\ \ref{sec:ghconj-modal}) so they cannot be analyzed as the being the same prefix.
Because these two prefixes are homophonous, unmarked aspect forms with the \fm{\Mf{g̱}-} modal prefix look identical to corresponding forms with the \fm{\Af{g̱}-} conjugation prefix.
But there are still crucial semantic and paradigmatic differences between modal \fm{\Mf{g̱}-} and conjugation \fm{\Af{g̱}-}.
Semantically, the \fm{\Mf{g̱}-} modal prefix only occurs in forms that express possibility or necessity which is not necessarily the case for forms with the \fm{\Af{g̱}-} conjugation prefix.
Paradigmatically, the hortative and contingent mentioned above are never found with the \fm{\If{i}-} stative prefix.
The tables in this section therefore have forms that are similar to those in section \ref{sec:ghconj} for the \fm{\Af{g̱}-} conjugation prefix, but the possible combinations of prefixes are different.

\subsection{Unmarked aspect with irrealis \textit{u-} and modal \textit{g̱-}}\label{sec:zero-irrealis+modal}

\begin{multicols}{2}
\noindent
The combination of irrealis \fm{\Rf{u}-} and modal \fm{\Mf{g̱}-} is possible with unmarked aspect.
Since the imperfective aspect does not occur with modal \fm{\Mf{g̱}-}, it is not a surprise that the addition of \fm{\Rf{u}-} to \fm{\Mf{g̱}-} is also impossible for imperfective aspect.
All unmarked aspect forms with irrealis \fm{\Rf{u}-} and modal \fm{\Mf{g̱}-} are therefore \fm{∅}-conjugation class verbs where the prefix for conjugation class is required.
As noted earlier in subsection \ref{sec:zero-modal}, the only context for all three of irrealis \fm{\Rf{u}-}, the \fm{∅} conjugation prefix, and modal \fm{\Mf{g̱}-} is the potential modality.
All prefix combinations in this subsection therefore represent a subset of potential modality forms.
The examples in (\ref{ex:zero-irrealis+modal-forms}) illustrate one \fm{∅}-conjugation verb with potential modality, namely the verb \fm{as.ée} (\fm{∅}; \fm{-μμH} activity) ‘s/he/it makes him/her/it become cooked’.

Potential modality forms normally include the stative \fm{\If{i}-} classifier prefix as seen in (\ref{ex:zero-irrealis+modal-forms-pot}).
The only context where they are known to lack \fm{\If{i}-} is when they are marked for past tense (“decessive”) with the past suffix \fm{-ín}.
The combination of past tense with potential modality is shown in (\ref{ex:zero-irrealis+modal-forms-pot+past}).
We would expect subordinate clause forms of potential modality to also lack \fm{\If{i}-} but there are no attested examples and no records of testing this possibility.
Crucially, negation of a potential modality form does not suppress \fm{\If{i}-} unlike negation of other forms with \fm{\If{i}-}.
